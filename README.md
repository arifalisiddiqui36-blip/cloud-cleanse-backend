# Data Redundancy Removal System

## Project Overview

The Data Redundancy Removal System is a cloud-inspired data validation and duplicate detection application developed using Flask, MySQL, and React. The system validates incoming records, identifies duplicate entries, prevents redundant data from being stored, and ensures database accuracy.

## Features

* Email Validation
* Duplicate Email Detection
* Duplicate Phone Number Detection
* False Positive Identification
* Unique Data Storage
* Real-Time Record Management
* Interactive Dashboard
* Modern User Interface

## Technologies Used

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Loveable AI

### Backend

* Python
* Flask
* Flask-CORS

### Database

* MySQL

## System Workflow

1. User submits data through the Entry Page.
2. Flask backend validates input.
3. System checks for duplicate email and phone records.
4. Invalid or duplicate data is rejected.
5. Only verified unique records are stored in the database.
6. Records are displayed through Dashboard and Records pages.

## Project Objectives

* Identify redundant data.
* Classify false-positive entries.
* Prevent duplicate records.
* Improve database accuracy.
* Maintain efficient data storage.
